Hierarquia

O que é hieraquia no HTML?

É a relação organizacional entre as tags que compões um documento HTML.
Uma organização clara e semântica facilita a compreensão do conteúdo pelos usuários e pelos mecanismos de busca, melhorando a experiência do usário (UX) e o posicionamento da página (SEO) nos resultados de pesquisa.

Organização

Um HTML organizado precisa contar com algumas coisas:

  • Indentação
    • Organização visual que facilita na manutenção e compreensão do código;
  • Hierarquia
    • Ordem de de escrita otimizada do código;
  • Acessibilidade
    • Atributos adicionados em tags para garantir acessibilidade para pessoas com deficiência e leitores de tela.
      • O atributo alt na tag 'img', para definir uma descrição alternativa para a imagem, que é lida pelos leitores de tela;
      • O atributo tabindex em elementos interativos, para definir a ordem de navegação por teclado.
